# Security Policy

## Supported Versions
We actively maintain and monitor the latest stable release of this repository.

## Reporting a Vulnerability
If you discover a security vulnerability, please **do not open a public issue**. Instead, contact us privately via email: [security@example.com].

We will respond within 48 hours and work with you to resolve the issue.

## Security Best Practices
- Keep dependencies updated
- Avoid committing sensitive information
- Use code reviews for all changes
